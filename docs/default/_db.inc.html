db_query (line 63)

Executa uma query

Executa uma consulta ao banco de dados, retornando o resultset com os resultados obtidos.

  • return: O resultset da consulta executada
PDOStatement db_query (string $sql, [array $params = array()])
  • string $sql: A consulta sql a ser executada. É possível utilizar demarcadores ? para indicar a posiçao dos parametros que serao substituidos na consulta.
  • array $params:

    Um vetor com os parametros que serao utilizados na consulta. A ordem dos paramentos no vetor deve ser a mesma dos marcadores na consulta:

    1.     $sql = SELECT * FROM TABELA WHERE id=? AND data > ?;
    2.     $params = array(1, '24-02-1986 00:00:00');
    3.     db_query($sql, $params);

    No exemplo acima, os marcadores em $sql (?), serao substituidos pelos valores em $params, resultando na consulta:

    1. SELECT * FROM TABELA WHERE id=1 AND data > '24-02-1986 00:00:00';

    Note que os parametros sao escapados automaticamente.

db_update_record (line 192)

Atualiza um registro no banco de dados.

Esta função atualiza um registro existente na tabela informada. Os novos valores serão atualizados de acordo com os dados em $record.

O registro a ser atualizado e indentificado pelo argumento $primary_key.

  • return: True em caso de sucesso, false em caso de falha.
bool db_update_record (string $tabela, array $record, array $primary_key)
  • string $tabela: O nome da tabela onde esta o registro a ser atualizado
  • array $record: Um vetor com os campos e valores a serem atualizados. As chaves representam os campos, e os valores os novos valores do registro.
  • array $primary_key: Um vetor com um unico elemento, onde a chave e o nome da coluna da chave primaria e o valor e o valor da chave primario do registro a ser atualizado.
